TEXT ADVENTURE PROJECT

Computer Science Principles

January 9, 2019 - January 16, 2019

↩ Go back



The objective of this project was to create a text adventure game using Python. We had a few guidelines to follow as well:

  • Multiple Acts
  • Branching Storyline
  • No Loops
For this project, my partner and I divided up the coding work for each of the acts. I coded acts 2 and 3, and my partner coded act 1. We worked collaboratively to develop the story and flow charts, which can be viewed here. This project was really fun for me because it allowed me to combine both my passion for my more artistic interests including storytelling and game design as well as my technical knowledge of the Python language and coding. It allowed me to be creative in ways that previous projects hadn't, because we had so few guidelines and restrictions. Throughout the course of this project, I learned the value of dividing up coding work becaues it allowed my partner and I to get a lot more done with each of us working on separate things at once and checking in when necessary. I also learned about how to code this specific type of game in Python: one with user input and choices that affect the storyline.

To download the Python file for the game, click here.

To run the game:
On Windows, open a Command-Line window, navigate to the correct directory using the cd command, and type in python Lafosse_Wang_Text_Adventure.py.
On Mac, first install Python. Then open Terminal, navigate to the correct directory using the cd command, and type in python Lafosse_Wang_Text_Adventure.py.
