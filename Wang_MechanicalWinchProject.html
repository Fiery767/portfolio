MECHANICAL WINCH PROJECT

Principles of Engineering

October 16, 2017 - October 20, 2017

↩ Go back


The objective of this project was to design, create, and test a small scale prototype of a mechanical winch able to lift an object of 100 grams up 30 centimeters, as well as create a NetLogo simulation of it. For this project, I was the primary NetLogo programmer. NetLogo is an agent-based language and integrated modeling environment. I furthered my knowledge of the NetLogo language, specifically programming movement of turtles (turtles are NetLogo's agents, which move around in the created world). Throughout the project, I encountered many small coding challenges, which all helped me to further my understanding of the NetLogo language and its functionality. A specific challenge I faced was when I pressed 'go' on the interface, the weight would spin with the drum instead of only the drum spinning. I was able to solve this problem by creating code that would make only the Drum turtle spin when the go function was called, and also by restricting the weight's movement to a single x-coordinate. Solving this problem and others helped me better understand how the NetLogo language functions.
