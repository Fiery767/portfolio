GREENFOOT EDUCATIONAL GAME PROJECT

AP Computer Science Applications

September 9, 2019 - September 20, 2019

<< Go back



The objective of this project was to modify an existing game in the Greenfoot IDE and convert it into an educational game. I worked in a team of 4, and we converted a simple breakout game into an educational chemistry game about the elements of the periodic table. The modifications I coded were:

  • Convert the bricks into the elements of the periodic table (and instantiate them in the correct locations)
  • Add subclassing for the different types of elements
  • Add sound effects based on the subclass of the element hit
  • Add a question for when all the elements of a subclass are destroyed
Unfortunately due to the time constraint, we were unable to add all the features we wanted to, and there were some bugs in the game when we were finished. For instance, all the columns from Boron down to the right end of the table don't have subclasses and thus don't have sound effects. There is also a bug that causes the game to ask you questions about elements you've destroyed in a column when you haven't actually destroyed any yet. If we had additional time, we would have fixed these issues and added a greater variety of questions, including questions addressing more advanced chemistry concepts.
