GREENFOOT INTERACTIVE SEATING CHART

AP Computer Science Applications

August 22, 2019 - September 9, 2019

<< Go back



The objective of this project was to use the Java Greenfoot IDE to create a profile that when combined with the rest of the class, would create an interactive seating chart. Additional requirements were that your profile, when clicked, would say your name, and perform some kind of animation, before returning to your seat and activating a "questions" mode. This project was our first introduction into the Greenfoot IDE, which we would later use in the Educational Game Project.
